# Announcements

## Document Purpose
This document contains realistic sample announcements for BrightPath English Learning Center. Announcements are included in the knowledge base so the AI Parent Assistant can answer questions about upcoming dates, parent actions, workshops, assessment periods, portal maintenance, safety reminders, and policy updates.

Announcements may change over time. The AI Parent Assistant should always treat the most recent relevant announcement as the primary source when answering date-specific questions. For official decisions, parents should contact the center directly.

## Announcement Interpretation Rules for AI
- If a parent asks about a date, include the date from the announcement.
- If a parent asks what to do, provide the parent action list in simple language.
- If a parent asks for an exception, explain that exceptions must be handled by center staff.
- If a parent asks about a specific student's eligibility, use authorized student data if available or recommend contacting the teacher or admin team.
- If announcements conflict, prioritize the newest announcement or instruct the parent to contact the center.
